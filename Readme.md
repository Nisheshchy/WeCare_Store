# WeCare Store

A Python-based inventory management system designed to manage product data using simple and reliable file handling. The project focuses on clean structure, modular design, and practical store operations.

## Overview

WeCare Stores manages product inventory through a console application. It stores product records in a text file and processes them using separate modules. The design keeps logic organized and easy to maintain.

## Core Features

- Product listing with details
- Stock tracking and updates
- Add and manage product records
- File-based data storage
- Modular architecture for scalability
